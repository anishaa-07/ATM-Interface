# ATM-Interface
A Java-based ATM system implementing real-world banking operations using OOP principles.

A console-based ATM simulation developed using Core Java and Object-Oriented Programming concepts. The application authenticates users using a User ID and PIN, then allows them to perform essential banking operations through a menu-driven interface. The project is implemented in a single Java file for simplicity and easy execution.
